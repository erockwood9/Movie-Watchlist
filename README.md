# Project Setup

This project consists of a **frontend** and **backend** that run together during development.

## Prerequisites

- Node.js (recommended: v18 or higher)
- npm (comes with Node.js)

## Installation

Install dependencies for both the frontend and backend:

```bash
npm run install:all
```

This command installs all required packages inside the `frontend/` and `backend/` directories.

## Running the Project

Start both the frontend and backend development servers:

```bash
npm run dev
```

This will run both services concurrently.

## Accessing the App

Once the servers are running, open your browser and go to:

```
http://localhost:5173
```
